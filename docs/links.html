MixedPsy

Alessandro Moscatelli and Priscilla Balestrucci are developer and maintainer of the R package MixedPsy: tools for the analysis of psychophysical data in R. The package is described in detail in the following reference:

Balestrucci, P., Ernst, M. O., Moscatelli, A. (2022). Psychophysics with R: The R Package MixedPsy. bioRxiv, 2022-06.
